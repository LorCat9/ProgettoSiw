Progetto Sistemi Informativi Su WebProgetto Avanzato

Si vuole realizzare il sistema informativo su Web per la gestione di una piccola galleria d’arte.

Possono accedere al sistema utenti generici e un amministratore.L’amministratore, previa autenticazione, può inserire e rimuovere le informazioni relative alle opere d’arte presenti nella galleria; per semplicità supponiamo che le opere d’arte siano quadri e che per ciascuno di essi siano di interesse il titolo, l’anno in cui è stato realizzato, la tecnica, le dimensioni, l’autore.

Dell’autore sono di interesse nome, cognome, nazionalità data di nascita e data di morte.L’utente generico può accedere alle informazioni della galleria attraverso diversi percorsi di navigazione, opportunamente predisposti (ad esempio, per autore, oppure per anno, etc.).

Completare ed estendere le specifiche a piacere dove si ritiene necessario.Progettare il sistema, definendone casi d’uso, class diagram (con indicazioni utili alla progettazione dello strato di persistenza), schema della base di dati. Implementare almeno 4 casi d’uso.
